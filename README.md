# vue空工程
做了部分插件安装的初始化Vue工程

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### mock data by json-server
```
npm run mock
```

### request by Axios
```
this.$get(url, params).then(res => {})
this.$post(url, params).then(res => {})
```
